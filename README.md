# 수능 비문학 생성기 노트

## 생성 기준

### 지문 및 문항 분석
지문 처리
- 인문 사회 과학 기술 예술 등 비문학 전 영역 지문 대상
- 문단별 핵심 쟁점 및 논거 결론 서술 방식 구조 파악
- 중요 구절 마킹
  - 의미 해석 중요한 지점에 기호(㉠ ㉡ ㉢) 및 `<밑줄>` 태그 자동 삽입
  - 글쓴이 의도나 핵심 주장 담긴 문장 우선 선택
- 2015 개정 교육과정 및 2024학년도 수능 출제 기준 준수

문제 유형 14종
- 내용 이해 (사실 추론 의도 밑줄의미)
- 글의 구조 (전개패턴 논증구조)
- 비교와 평가 (개념비교 관점비교 타당성평가)
- 적용과 확장 (원리적용 복합자료해석 문제해결)
- 어휘력 (문맥의미 개념이해)

난이도 보정
- 상 복잡한 다단계 추론 및 암시적 의미 파악 종합 정보 분석 능력 측정
- 중 기본적인 추론 및 명시 암시 내용 균형 잡힌 이해도 평가
- 하 지문 명시 정보 직접 확인 및 단순 사실 관계 파악 중심

### 선택지 구성 및 해설 로직
선지 설계 원칙
- 기본 형식 ①-⑤ 5지 선다형 구성
- 정답 구성 지문 내 명확한 텍스트 근거 및 논리 인과관계 필수
- 오답 설계 기법 (과잉일반화 부분적이해 인과오류 맥락무시 등)

데이터 구조
- 정답 데이터 1~5 정수형 매핑
- 상세 해설 (정답 도출 경로 및 오답별 논리 오류 구체적 기술)

## 주요 기능

### 문항 생성
- 단일 모드 특정 유형 및 난이도별 설정 적용
- 배치 모드 여러 유형 문항 동시 호출 일괄 생성 및 통합 마킹 제공

### 인터페이스
- 표시 섹션 원본 지문 스트리밍 및 마킹 지문 렌더링 문제 카드 인터랙션
- 결과 제어 정답 해설 토글 및 생성 문항 피드백 표시

## 기술 스택

### Core
- Next.js 15 (App Router) TypeScript
- Tailwind CSS v4 Radix UI
- Zod Lucide-React JSONWebToken

### 연동
- 출제 기준 2024학년도 수능 국어 출제 매뉴얼 가이드라인
